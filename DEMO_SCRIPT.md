## Recording Tips
- Use **OBS** or **Screen Studio** — 1080p, 60fps
- Zoom into important elements (scores, AI summary, security findings)
- Use Cmd/Ctrl+F to fullscreen the browser
- Pause briefly at each result screen so viewers can read
- Total target: **55-65 seconds**
